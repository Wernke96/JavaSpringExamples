## Quick Start Guide

1. Change directory into ./nku `cd nku`

2. Install Dependencies: `mvn install -D skipTests`

3. Run the web application: `java -jar target/nku-0.0.1-SNAPSHOT.jar`

4. Open a web browser and go to: http://localhost:8080/
